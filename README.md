## Overview

This minimal Cloudflare Workers starter contains a single REST API endpoint (`GET /`) that returns a JSON payload with a message, a timestamp, and the client's IP address.

## Tech Stack

### Backend

- **Language**: [JavaScript](https://tc39.es/ecma262/multipage/)
- **Serverless**: [Cloudflare Workers](https://workers.cloudflare.com/)

### Quality Assurance

- **API testing**: [Vitest](https://vitest.dev/)
